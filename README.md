# nis2scan

NIS2 Article 21 compliance scanner. Runs on your 
infrastructure. No cloud. No subscription.

## What it does
Scans your Linux and Windows servers against all 
NIS2 Article 21 controls and produces an 
auditor-ready PDF report.

## Sample Reports
Two real scan reports are included in this repository:
- nis2scan_linux_45score.pdf — Ubuntu server scan
- nis2scan_windows_10score.pdf — Windows Server 2025 scan

## Supported platforms
- Ubuntu / Debian Linux
- Windows Server 2019 / 2022 / 2025

## How it works
1. Configure config.yaml with your server details
2. Run ./nis2scan --config config.yaml --progress
3. Get PDF report
